#### 기본으로 들어있는 스타일 제거

- 기본적으로 html, body에 margin이 들어있다.
- 그대로 나두게 되면 꽉찬 화면을 만들 수 없다.

- body,html이 전체화면 채우도록 크기를 지정해준다.
